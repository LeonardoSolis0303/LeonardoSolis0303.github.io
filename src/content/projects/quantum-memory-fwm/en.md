## Summary

My undergraduate thesis at LAFriOC (Instituto de Física, UNAM), advised
by Dr. Daniel Sahagún. The project explores four-wave mixing in cold atoms
as a quantum memory system — that is, as a substrate for storing and
retrieving photonic states.

## Problem

A quantum memory must do three things: accept an incoming photon, store
its quantum state coherently, and re-emit it on demand without
collapsing the encoded information. Cold-atom four-wave mixing is one of
the protocols where this can be done with high fidelity, in principle.

## Method

- Cold-atom setup based on rubidium, with magneto-optical trapping
  using Helmholtz coils and laser cooling.
- A four-wave mixing scheme that couples write and read fields to the
  atomic ensemble.
- Characterization of write efficiency, storage time, and retrieval
  fidelity.

This is partly an instrumentation thesis: a non-trivial fraction of the
work is electronics, optics, and the boring infrastructure of running
the experiment well.

## Status & Results

Ongoing. Currently characterizing the magnetic field stability and the
fast-switching circuits that gate the trapping fields. No published
results yet.

## Learnings

- Designing the apparatus is half the experiment. The other half is
  realizing how many things can be wrong with the apparatus.
- Numerical simulation of the four-wave mixing dynamics has been useful
  for setting expectations, but no substitute for actually running the
  thing.

There is no public repository yet. I will share simulation code and
characterization data once the thesis is closer to defense.
